## 1 Installation
From the current directory (which includes this `README.md`), run
```
cd ikd
pip install -e .
```

## 2 Tutorial
`demo.ipynb` in `ikd` is a step-by-step tutorial that run PCA, GPLVM, and IKD on a synthetic dataset.
